# Assisi
* Biphasic DAQ with ADC done cDAQ hardware

## V0.1.0
* Initial Assisi PCB version copied forward from earlier circuitry.
* Used as a holdover design to preserve and test the core circuit layout.

## V0.2.0
* First version focused on adapting the DAQ PCB for the enclosure.
* Began moving the design toward a boxed, deployable hardware package.

## V0.2.1
* Fixed issues found in V0.2.0 and added enclosure bolt holes.
* Printed enclosure version sent to Italy in June 2026.

## V0.2.2
* Shortened the PCB and added a notch/cutout for better enclosure fit.
* Cleaned up the schematic, PCB traces, and moved toward two mounting holes.

## V0.2.3
* Future version to explore direct 5 V power input without the Arduino regulator.
* Planned update to include integrated resistor management.
